# 更新日志

## [2025-12-07] 核心架构重构与UI体验升级

### 🚀 新增功能与架构升级
*   **动态应用管理 (Apps Hot-Plugging)**
    *   **教师版**：启动后自动解压出 `apps` 目录。用户可在此目录中自由增加、删除或修改应用文件夹，重启软件即可生效。
    *   **学生版**：
        *   **自动携带**：导出时自动打包 `apps` 目录下的当前所有应用。
        *   **配置持久化**：程序结束时自动解压并保存各应用的配置文件（如 JSON 配置），实现用户数据的持久化存储。
        *   **源码保护**：仅解压配置文件，不会解压 Python 源代码文件，保护核心逻辑。
        *   **智能屏蔽**：自动检测并屏蔽“示例应用”，仅向学生展示实际教学工具。
    *   **一键重置**：若需恢复初始状态，只需删除 `apps` 目录后重启软件即可。
*   **标准化示例应用**
    *   新增 `示例应用`，作为开发规范的参考模板。

### 🎨 界面与交互优化
*   **统一视觉风格**
    *   全面重构了四大核心应用（图解法求理论板数、塔板负荷性能图、板式塔结构简图、阀孔实际阀孔数），实现了统一的界面布局和交互逻辑。
*   **画图页面增强**
    *   **可调分割线**：新增左右区域分割线，支持拖拽调整参数区与绘图区的宽度比例。
    *   **自适应滚动条**：参数配置区域新增垂直滚动条，有效解决了在低分辨率屏幕或缩放模式下参数显示不全的问题。
    *   **全屏铺满展示**：生成的图表现在会完全铺满右侧展示区域，利用率更高。

### ⚡ 性能优化
*   **渲染机制改进**
    *   **按需生成**：取消了参数修改时的实时渲染机制，改为“点击生成”触发。
    *   **性能提升**：大幅减少了不必要的重绘计算，显著降低了算力消耗，解决了调整参数时的卡顿问题。
    *   **高清输出**：支持按特定高分辨率生成图片，确保细节清晰。
